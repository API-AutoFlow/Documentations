# Current Time
Returns the current time in UTC.

## Properties
```yaml
Time-unit: Select from second, millisecond, microsecond
```

## Output
```yaml
Output-location: The location to store the timestamp
```

### Format
```js
hh:mm:ss.decimal (eg 19:20:30.40)
```
