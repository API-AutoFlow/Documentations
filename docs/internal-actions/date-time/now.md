# Now
Returns the current date and time in UTC.

## Properties
```yaml
Time-unit: Select from second, millisecond, microsecond
```

## Output
```yaml
Output-location: The location to store the timestamp
```

### Format
```js
YYYY-MM-DD hh:mm:ss.decimal (eg 1997-07-16 19:20:30.40)
```
